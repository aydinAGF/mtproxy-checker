📡 Proxy Scanner & Manager
Proxy Scanner is a lightweight, powerful Android utility designed to help you discover, manage, and test network proxies in real-time. Whether you are looking for secure ways to bypass restrictions, testing network latency, or just trying to find the fastest connection for Telegram, this app provides a streamlined all-in-one toolkit.
✨ Key Features
Multi-Protocol Support: Seamlessly parses and handles MTProto, SOCKS5, and standard HTTP proxies.
Flexible Import Sources:
Fetch proxies directly from Telegram channel links or web URLs.
Paste bulk proxy lists instantly from your clipboard.
Import proxies from local .txt files.
Smart Parsing Engine: Automatically extracts proxy servers, ports, and authentication credentials (usernames/passwords/secrets) from raw text, tg:// deep links, and standard URL formats.
Real-Time Latency Testing: Includes a built-in ping checker that actively tests the connection to each proxy, verifying if it is alive and displaying the exact response time in milliseconds.
One-Tap Telegram Integration: Instantly apply MTProto and SOCKS5 proxies to your Telegram app with a single click, or copy the configuration to your clipboard for use in other applications.
Modern UI: Built natively with Jetpack Compose, featuring a clean, responsive, and dark-themed interface perfect for power users.
🚀 How it Works
Simply add a source URL or paste a list of proxies into the app. The engine will instantly parse all available configurations and begin testing their connectivity. Once the checks are complete, you can tap the "Send" icon to immediately open the working proxy in Telegram!
